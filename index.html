Introduction

Loris is a set of patterns and utilities for building a beautiful RESTful JSON API. It aims to integrate easily into complex data backends while simplifying access and providing discovery tools for developers.

For complex relational data

Loris was built to quickly turn relational data into a searchable, queryable RESTful API. It makes no assumptions on the structure of the data store, but instead supplies tools and patterns for implementers to easily build an API frontend.

Optimizing data access

Like all modern APIs, Loris performs request caching for idempotent operations, dramatically reducing the load on the data store. Loris also reduces network traffic by optionally allowing retrieval of related resources to occur within the same request.

Simple access controls

Includes support for fine-grained access control on individual resources. By using simple access control lists, Loris aims to be easily integrated into existing access control schemes.

Robust searching

Using the well established Elasticsearch as a backend, all data is searchable and filterable with near instant results. Loris provides additional tools to filter search results by access controls, scoping search results to the individual.

Focus on analytics

Loris puts analytics topmost for requirements, aiming to provide useful information for developers to analyze requests to the API, as well as information for backend implementers to help optimize where it matters most.
